Firefox

Rychlý a bezpečný webový prohlížeč, o kterém všichni tak nadšeně mluví.

Firefox Focus

Firefox Focus je jednoduchý prohlížeč pro Android a iOS. Nabízí automatické blokování reklam a ochranu proti sledování.

Česká komunita

Na stránkách české komunity Mozilly, najdete informace o aktuálním dění a můžete se zde sami do projektu zapojit.

Podpora uživatelů

Stránky a fórum podpory Firefoxu obsahují řadu tipů a návodů, které vám mohou pomoci. Pokud máte problém s Firefoxem, je to místo určené právě vám.

Doplňky pro Firefox

Rozšiřte si Firefox podle svých představ pomocí nepřeberného množství doplňků. Vytvořte si svůj Firefox!
